## Overview

Professional-grade network port scanner for authorized security assessments. Features multi-threaded scanning, service fingerprinting via banner grabbing, multiple export formats, comprehensive logging, and full unit test coverage. Designed for ethical security research and network auditing only.

## Features

- **Multi-threaded Scanning** — Configurable thread pool for fast results
- **Banner Grabbing** — Service fingerprinting and version detection
- **Export Results** — CSV, JSON, and HTML report formats
- **Comprehensive Logging** — Structured logging with timestamps
- **Unit Tests** — Full test coverage with pytest
- **Secure Coding** — Input validation, timeout handling, resource cleanup

## Quick Start

```bash
git clone https://github.com/Raphasha27/Network-Port-Scanner.git
cd Network-Port-Scanner
pip install -r requirements.txt
python scanner.py --target example.com --ports 1-1024
```
